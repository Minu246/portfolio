Projects

ONLINE COFFEESHOP SYSTEM

The Online-Coffee Shop System is based on a concept to maintain the table booking, food ordering and applying job in to that particular coffee shop. There are four sections in this project admin panel, table booking, food ordering and applying job. In this system two type of users such as admin and user. The users are two types one of them is job seekers and other is customers. By selecting coffee order the system displays a list of available drinks and foods the user has to place an order with item quantity. After that, he/she proceeds towards order confirmation and payment methods. Online-Coffee Shop System in ASP.NET consists following modules such as login module, customer module, administrator module. This project is developed in ASP.NET using C# language and MSSQL database is used. The role of the administrator is to maintain customer information including operations like modifying, deleting, updating the customer records, customer order records and job application in the system.
